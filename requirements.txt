# driver: check with `nvidia-smi` (top-right shows max supported CUDA).
#

#    pip install torch --index-url https://download.pytorch.org/whl/cu121
#
# 2) PyTorch Geometric + its companion wheels (must match torch & CUDA):
#    pip install torch_geometric
#    pip install pyg_lib torch_scatter torch_sparse \
#        -f https://data.pyg.org/whl/torch-2.3.0+cu121.html
#    (replace 2.3.0+cu121 with your actual `python -c "import torch;print(torch.__version__)"`)
#
# 3) Utilities:
#    pip install h5py pyyaml numpy tqdm scikit-learn
#
# 4) ONLY if you parse STEP files yourself (skip if using the prebuilt H5 graphs):
#    conda install -c conda-forge pythonocc-core=7.7.0
